## The Question

"Poverty leads to drug problems" is a widely repeated assumption that is
rarely tested directly. This project asks a narrower, testable version:
**across countries, is a nation's income level statistically associated
with its drug-use death rate?**

The analysis is at the **country level** for 2019 (the last complete
pre-COVID year) — a test of an aggregate, structural pattern, not a claim
about individuals.

## Five Hypotheses

| | Test | H₀ | Result |
|---|---|---|---|
| H1 | Correlation (Pearson & Spearman) | correlation = 0 | r = +0.416, ρ = +0.473, both p < 0.001 |
| H2 | Two-sample t-test (richer vs. poorer half) | equal mean mortality | t = 3.77, p < 0.001, d = 0.56 |
| H3 | One-way ANOVA (6 world regions) | equal mean across regions | F(5,179) = 4.34, p < 0.001, η² = 0.11 |
| H4 | Linear regression on log(GDP) | slope = 0 | slope = +0.67, p < 0.001, R² = 0.128 |
| H5 | Robustness check (US excluded) | results unchanged without US | all results hold; several strengthen |

**Headline finding:** higher income is associated with *more* recorded
drug-use deaths, not fewer — the opposite of the common assumption. This
is **association, not causation** — the analysis is ecological and
cannot be applied to individuals. Reporting-capacity differences between
rich and poor countries, and the US opioid crisis specifically, are the
leading explanations (discussed in the presentation, not proven by it).

## How to Reproduce

Requires R ≥ 4.3 and the following packages, all installable via
`install.packages()` or (on Debian/Ubuntu) `apt install r-cran-<name>`:
`dplyr`, `tidyr`, `readr`, `stringr`, `ggplot2`, `scales`, `lmtest`, `sf`,
`ggrepel`, `e1071`, `patchwork`.

```r
setwd("scripts")
source("01_clean_and_merge.R")   # -> ../output/analysis_ready.csv
source("02_analysis.R")          # -> prints every statistic (or redirect to results.txt)
source("03_make_charts.R")       # -> ../output/*.png (7 chart files)
```

Each script depends on the previous step's output, so run them in order.
`analysis_ready.csv`, `results.txt`, and the chart PNGs are already
included in `output/` for convenience.

## Methodology Notes

- **Preprocessing:** both sources filtered to real countries (3-letter
  ISO code, dropping regional aggregates), restricted to 2019, merged on
  ISO code, rows with missing values dropped. Final sample: 185 countries
  (from 4,422 WHO rows and 7,240 World Bank rows, all years).
- **Why log(GDP):** raw GDP per capita is heavily right-skewed (skewness
  ≈ 1.68); logging it brings skewness to ≈ −0.25, which is why the
  regression uses log(GDP) rather than raw GDP.
- **Why both Pearson and Spearman:** Pearson tests the linear
  association; Spearman tests rank-order association and is robust to
  outliers — reported together so the finding doesn't depend on one
  assumption.
- **Regression assumptions:** linearity, independent observations,
  normal residuals, and constant variance are all tested (Shapiro–Wilk,
  Breusch–Pagan). The latter two fail here, traceable to the United
  States being an extreme outlier — directly re-examined in the
  robustness check (H5).
- **Quartile binning:** both GDP and drug-death rate are binned into
  quartiles for the world-map visualization (course technique, binning a
  quantitative variable into a categorical one).
